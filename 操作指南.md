# A股交易预案应用 - 操作指南

## 项目结构

```
Stra-UPUP/
├── App.js          # 应用主代码（React Native）
├── app.json        # Expo配置文件
├── package.json    # 项目依赖配置
├── .gitignore      # Git忽略文件
└── assets/         # 资源文件目录
```

## 前置条件

- Node.js 已安装在 `D:\Program Files\nodejs`
- 项目文件已创建完成

## 操作步骤

### 1. 打开命令提示符

按 `Win + R`，输入 `cmd`，回车打开命令提示符。

### 2. 进入项目目录

```cmd
cd "C:\Users\Kobe\Desktop\策略器\Stra-UPUP"
```

### 3. 安装项目依赖

```cmd
"D:\Program Files\nodejs\npm.cmd" install
```

等待安装完成，这可能需要几分钟时间。

### 4. 安装 Expo CLI（全局安装）

```cmd
"D:\Program Files\nodejs\npm.cmd" install -g expo-cli
```

### 5. 启动开发服务器

```cmd
"D:\Program Files\nodejs\npx.cmd" expo start
```

启动成功后，会显示一个二维码。

### 6. 在华为P70上安装和运行

#### 方法一：使用 Expo Go（推荐，开发测试）

1. 在华为P70上打开应用商店
2. 搜索并下载 **Expo Go** 应用
3. 打开 Expo Go 应用
4. 扫描电脑上显示的二维码
5. 应用将自动加载并运行

#### 方法二：生成 APK 安装包（正式发布）

1. 注册 Expo 账号（访问 https://expo.dev ）
2. 登录 Expo 账号：
   ```cmd
   "D:\Program Files\nodejs\npx.cmd" expo login
   ```
3. 构建 APK：
   ```cmd
   "D:\Program Files\nodejs\npx.cmd" expo build:android
   ```
4. 等待构建完成，下载 APK 文件
5. 将 APK 文件传输到华为P70并安装

## 应用功能

### 主要功能

1. **新建交易预案**
   - 日期选择
   - 国家意志（政策方向、监管动态）
   - 消息面（行业新闻、公司公告）
   - 资金类型/对手盘分析
   - 大盘分析
   - 板块分析
   - 个股分析
   - 买入条件
   - 卖出条件
   - 备注

2. **历史预案管理**
   - 查看历史预案列表
   - 编辑历史预案
   - 删除历史预案

### 使用流程

1. 打开应用，进入"新建预案"页面
2. 填写各个维度的分析内容
3. 设置买入和卖出条件
4. 点击"保存预案"按钮
5. 在"历史预案"页面查看和管理已保存的预案

## 常见问题

### Q1: 安装依赖时出现网络错误

**解决方案：**
- 使用淘宝 npm 镜像：
  ```cmd
  "D:\Program Files\nodejs\npm.cmd" config set registry https://registry.npmmirror.com
  ```
- 然后重新运行 `npm install`

### Q2: Expo Go 无法扫描二维码

**解决方案：**
- 确保手机和电脑在同一WiFi网络下
- 手动输入 URL 地址
- 检查防火墙设置

### Q3: 应用启动后白屏

**解决方案：**
- 检查 App.js 文件是否有语法错误
- 查看终端输出的错误信息
- 重新启动开发服务器

### Q4: 华为P70无法安装APK

**解决方案：**
- 开启"允许安装未知来源应用"设置
- 路径：设置 → 安全 → 更多安全设置 → 安装外部来源应用
